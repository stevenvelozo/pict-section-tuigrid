# Pict Section TuiGrid

> Configurable data grids for Pict applications

Display and edit tabular data with the Toast UI Grid wrapped as a Pict view. Define columns, formatters, and editors in a JSON configuration object â€” or extend the class to override data marshaling for computed columns, custom validation, and live solver integration.

- **Config-Driven** -- Define column schemas, editors, and formatters entirely in JSON
- **Built-In Editors** -- Text, number (with decimal precision), select dropdowns, and date pickers
- **Formatters** -- Currency, rounded numbers, two-digit decimals, and date formatting out of the box
- **Solver Integration** -- Mark columns that trigger the Pict application solver on change
- **Extensible** -- Override `preChangeHandler`, `changeHandler`, and `customConfigureGridSettings` for custom behavior
